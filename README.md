# DataScience (Morning) Batch

This repository contains all learning materials, lecture notes, projects, and assessments for the **DataScience (Morning) Batch**, instructed by **Uday**.

---

## Curriculum

The batch covers the complete foundation and core areas of Data Science, Machine Learning, and Artificial Intelligence.

**Core Libraries**

* NumPy
* Pandas
* Matplotlib
* Seaborn

**Mathematics & Statistics**

* Descriptive statistics
* Probability fundamentals
* Linear algebra essentials
* Calculus basics for machine learning
* Statistical inference

**Machine Learning**

* Supervised learning
* Unsupervised learning
* Model evaluation
* Feature engineering
* Model optimization

**Deep Learning & AI**

* Neural networks fundamentals
* Deep learning concepts
* AI fundamentals and applications
